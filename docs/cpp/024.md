# 2.4 const 限定符

```c++
const修饰指针
int a = 10;
int b = 20;
const int *p = &a;//const 修饰 int
*p = 20;//错误
p = &b;//正确
int *const p2 = &a;//const 修饰指针
*p2 = 100;
p2 = &b;
```

## 2.4.1 const 的引用

```c++
const int & ref=10;//等价于 int temp=10;const int & ref=temp;
```

## 2.4.2 指针和 const

### const 指针

```c++
int & ref=a;

ref=10;
```
