# String 类

String 类是个 final 类，不能被其他类继承。

```java
 public static void main(String[] args) {
        String name = "NU";
        name = "tom";
        final char[] value = {'a', 'b', 'c'};
        char[] v2={'a','b','c'};
        value[0] = 'd';
        value=v2;//报错，因为 value 是 final 的，不能改地址。
    }
```
