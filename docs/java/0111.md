# 1.11 HashSet

```java
public static void main(String[] args) {
        Set set = new HashSet();
        set.add("join");
        set.add("lucy");
        set.add("join");
        set.add("jack");
        set.add(null);
        set.add(null);
        System.out.println("set=" + set);
    }
```

HashSet 是无序的，不能重复的。

HashSet 实际上是 HashMap。
