# 4.4 CISC 和 RISC 的基本概念

## 4.4.1 复杂指令集系统计算机（CISC）

X86

## 4.4.2 精简指令系统计算机（RISC）

ARM、MIPS

|            | CISC                 | RISC                  |
| ---------- | -------------------- | --------------------- |
| 指令系统   | 复杂                 | 简单                  |
| 可访存指令 | 不加限制             | 只有 Load、Store 指令 |
| 控制方式   | 微程序控制           | ==组合逻辑控制==      |
| 指令流水线 | 可以通过一定方式实现 | 必须实现              |
| 指令长度   | 不同                 | 相同                  |
