### 什么是驱动表？

多表关联查询时，第一个被处理的表就是驱动表，使用驱动表去关联其他表。

## 三种 join 算法

简单的嵌套循环连接

```sql
select * from user t1 left join order t2 on t1.id=t2.user_id
```
