# 3.4 流量控制与可靠传输机制
## 3.4.1流量控制、可靠传输与滑动窗口机制
滑动窗口就是流量控制和可靠传输（发送方超时重传）
### 3.可靠传输机制
确认
重传

信道利用率=发送时间/总时间=（L/C）/T

发送窗口大小<=窗口总数-1
## 3.4.2 停止-等待协议
发送窗口大小=1，接收窗口大小=1
## 3.4.3 后退N帧协议
累计确认，按序接收 

发送窗口大小>1，接收窗口大小=1

某一帧确认表示之前的帧都正确收到

发送窗口大小
$1\leqslant W_{T}\leqslant 2^{n}-1$，因为要区分新帧和旧帧
## 3.4.4 选择重传协议
发送窗口大小>1，接收窗口大小>1

一定：$W_{T}+W_{R}\leqslant 2^{n}$

一般而言：$W_{Tmax}=W_{Rmax}=2^{n-1}$

相对后退N帧的优点：来者不拒
