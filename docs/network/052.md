分用：接收方的传输层剥去报文首部后，能够这些数据正确交付到目的进程

UDP 的差错校验只是检查数据在传输过程中有没有出错，出错的数据直接丢弃。

## 5.2.2 UDP 校验

校验和计算中计算 UDP 伪首部的理由：

TCP/IP中识别一个进行通信的应用需要5大要素：“源IP地址”、“目标IP地址”、“源端口”、“目标端口”、“协议号”。UDP首部只有“源端口”、“目标端口”，其余三项在IP首部中。为了可靠传输。
