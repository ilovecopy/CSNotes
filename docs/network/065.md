## 6.5.2 超文本传输协议 (HTTP)

HTTP 规定了浏览器怎样向万维网服务器请求万维网文档，以及服务器怎样把文档传送给服务器

[HTTP 是基于 TCP 还是 UDP 的？](https://www.zhihu.com/question/20085992/answer/2049378419)

### 1.HTTP 的操作过程

单击一个链接后发生了什么？

1. 浏览器分析链接🔗指向页面的 URL
2. 浏览器向 DNS 请求解析 www.baidu.com 的 IP 地址
3. 域名系统解析
4. 浏览器与该服务器建立 TCP 连接
5. 浏览器发出 HTTP 请求
6. 服务器发送响应报文
7. 释放 TCP 连接
